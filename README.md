# Mab_ariba
ariba library for predicting clarithromycin resistance in M. abscessus

see https://github.com/sanger-pathogens/ariba/wiki/Task:-prepareref

## To Setup:
ariba prepareref -f Mab_ref.fa -m Mab_ref.tsv Mab_ref.out
